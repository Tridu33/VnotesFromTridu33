> 另外一点比较庆幸的是，LaTeX目前具有很多在线系统，如**overleaf** (链接为[https://www.overleaf.com](https://link.zhihu.com/?target=https%3A//www.overleaf.com/))，只需要一个浏览器（如Chrome）就可以开始使用LaTeX，大大提高了便利性。
