# 语义子程序


群友问：

>如何理解语义和语法呢
语义是语言的特性吗



zz 回答：



>一个语言只有语义而没有语法，你就无法进行证明，而一个语言只有语法而没有语义，你就无法判断对错。  
从语法的可推出，到语义的有效，是谓可靠性。从语义的有效到到语法的可推出，是谓完全性也。
