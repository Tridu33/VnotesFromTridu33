# 魔法命令新建
•%timeit

多次执行一条语句，并返回平均时间，%%timeit->多条语句

• %time

返回执行一条语句的时间，%%time->多条语句

• %reset

删除当前空间的全部变量

• %run *.py

在IPython中执行Python脚本
