# 数据模型、领域模型和视图模型








[软件体系结构——4+1视图（整理资料）](https://blog.csdn.net/ananhao/article/details/78938960?utm_medium=distribute.pc_relevant.none-task-blog-baidujs-3&spm=1001.2101.3001.4242)

四个角色

1.  数据模型：面向持久化，数据的载体。
2.  领域模型：面向业务，行为的载体。
3.  视图模型：面向UI（向外），数据的载体。
4.  命令模型：面向UI（向内），数据的载体。
