# STL-Boost为什么hpp文件




模板的编译和链接
模板的实现文件和定义文件分开编写，这时在使用模板的时候，我们就不能像往常一样，包含模板的定义文件（.h），这样会报链接失败的。这时的正确的做法是xx.cpp包含 模板的实现文件(.cpp)而不是定义文件(.h)。




boost : 包含 .hpp 头文件
lib : 包含 .lib 文件(编译的实现细节)
libs :这些实现细节的来源
