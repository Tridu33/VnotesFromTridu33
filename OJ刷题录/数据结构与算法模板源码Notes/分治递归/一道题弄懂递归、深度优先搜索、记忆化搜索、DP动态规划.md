# 递归
递归都是这样子的


指数级别递归算法——>动态规划公式法，只算必要的数据，避免不必要的重复计算部分——>转换为多项式循环


信息守恒，其实递归就是利用系统栈，我们的循环代码就是利用独有的用户设计的保存需要信息的关键的信息。循环的代码本身就是Goto,就是用户Goto.

但是好处就是这样子比较快，省时间地点，特别是当问题规模很大的时候。
